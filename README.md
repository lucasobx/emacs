### Requirements
* Emacs 29+
* PragmataPro
* Nerd Icons
* libvterm
* ripgrep
* cmake
* libtool

### Emacs as Terminal
Create a system shortcut to launch Emacs in terminal mode with:

```bash
emacs --eval "(my/vterm-only)"
```
